
编辑器说明：

### 分成几列：
    1. 最多生成8列，最少1列。

### 颜色数量：
    1. 会根据当前瓶子的数量和水柱的数量，会限制颜色的数量上限。

### 前一关/后一关：
    1. 可以在编辑器中切换前一关/后一关。
    2. 加载的关卡数据会出现在上方的编辑框中（可以把关卡数据复制出来）。
    3. 新关卡默认会放三个空瓶子。

### 生成关卡：
    1. 会根据当前瓶子的数量和水柱的数量，会自动生成关卡数据。
    2. 生成的关卡数据会出现在上方的编辑框中（可以把关卡数据复制出来）。

### 每一列的数量和位置：
    1. 选择按钮'0','1','2','3','4'表示当前列可以放置的瓶子数量
    2. 选择‘+’，‘-’可以上下调整当前列中瓶子的位置。
    3. UiEdColumn.ts中可以设置每次移动的距离。
